Hexo搭建个人博客

1.安装前提

先安装下列应用程序：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

3.部署到GitHub

因为每次频繁的启动太麻烦可以把本地博客推送到GitHub上去，以完成静态博客远程服务端的部署。用户部署个人博客的GitHub仓库的命名必须符合特定要求才行 比如 xxxx.github.io

推送到仓库时需要安装 hexo-deployer-git 插件

完成后进行生成和部署

4.更换主题

前往官网https://hexo.io/themes/index.html 挑选自己喜欢的主题或者去GitHub上搜索hexo-theme来寻找

我使用的是 hexo-theme-yilia 主题

在博客下的 _config.yml 文件 找到theme改为你使用的主题名

更换好新主题最好先把之前的pubilc文件清除再生成新的之后再部署
